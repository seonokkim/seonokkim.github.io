# Riley Kim - Personal Website

This repository contains the source for my GitHub Pages site.

- Site: https://seonokkim.github.io
- Theme: Just the Docs

## Update the site

Edit `index.md` or other markdown pages and push to `main`.

GitHub Pages is configured to publish from:

- Branch: `main`
- Folder: `/(root)`

## Notes

This repository is for the published website, not for theme development.
